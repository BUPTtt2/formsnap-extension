## 视频一：15秒痛点直击版

**时长**：15秒
**风格**：快节奏、痛点共鸣

| 时间 | 画面 | 台词/字幕 |
|------|------|----------|
| 0-3s | 黑屏白字：你每天花多少时间填表？ | 你每天花多少时间填表？ |
| 3-6s | 屏幕录制：手动复制粘贴信息到各个表单 | 同样的信息，复制粘贴几十遍 |
| 6-10s | 屏幕录制：打开 FormSnap，截图上传，AI 自动识别，一键填写 | FormSnap：截图即填，一键搞定 |
| 10-13s | 屏幕录制：坚果云表格，生成 TSV，Ctrl+V 粘贴 | 连 Canvas 表格都能填 |
| 13-15s | 产品 Logo + 下载引导 | 免费安装，评论区见 |

**BGM**：节奏感强的电子/科技风

## 发布文案模板

**文案 1（痛点型）**：
同样的姓名身份证号银行卡号，在不同系统填了几十遍？
FormSnap 帮你截图即填，AI 自动识别一键搞定。
支持坚果云、问卷星、所有网页表单。
免费开源，评论区拿链接。

**文案 2（功能型）**：
发现一个神器：截图里的文字，AI 自动提取并填到表单里。
多张截图 = 多行数据，直接粘贴到在线表格。
还支持历史记录复用，下次填表直接调用。
程序员开源的，完全免费。

**文案 3（故事型）**：
实习生每天花 2 小时填表，直到同事推荐了这个插件...
三分钟搞定原来半小时的工作。
TRAE 大赛作品，技术硬核，体验丝滑。
