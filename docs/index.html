核心功能

三种输入，一键填写

无论你的数据在截图里、文本里还是 Excel 文件里，FormSnap 都能识别并自动匹配。

截图智能识别

上传任意截图，多模态 AI 自动提取字段名和值。支持多张截图同时解析，生成多行数据。

文本 & 文件导入

直接粘贴任意格式文本，或上传 Excel/CSV 文件。自动解析为结构化数据，无需手动整理。

Canvas 表格突破

自动检测 SpreadJS Canvas 表格，通过 AI 视觉识别列名，生成 TSV 数据一键粘贴到坚果云等在线表格。

三级智能匹配

精确匹配 + 模糊匹配 + AI 语义匹配。"成绩"自动对应"最佳成绩"，无需手动调整。

填写历史记录

每次填写自动保存，支持编辑、删除、复用。TSV/JSON/文本三种格式一键复制。

一键撤销

填错了？点击撤销，瞬间恢复表单原始数据，零风险尝试。

使用流程

三步搞定

01

输入数据

截图、粘贴文本或上传 Excel/CSV 文件。AI 自动识别所有字段和值。

02

确认匹配

AI 自动将数据源字段与表单字段匹配。拖拽调整，所见即所得。

03

一键填写

点击填充按钮，所有字段瞬间填写完成。Canvas 表格则复制 TSV 粘贴即可。

安装步骤

  1. 1 下载源码并运行 npm run build
  2. 2 打开 Chrome，访问 chrome://extensions/
  3. 3 开启「开发者模式」，加载 dist/ 目录
  4. 4 点击扩展图标，配置你的 API Key
